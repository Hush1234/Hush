소개

Root 동아리소개
Root 는 IT 관련 활동을 하는 자율 동아리입니다.
저희 동아리는 김성식 선생님이 담당 교사 선생님으로 계시며
저희 동아리에서는 웹페이지 제작, 아두이노 프로젝트 등에 다양한 IT 관련 활동을 하고있습니다.
저희 Root 웹 페이지는 소개, 최근 프로젝트, 기수별 학생 명단으로 나누어져 있으며
최근 프로젝트란에서 아두이노등 에 관한 활동 내용들을 확인할 수 있습니다.
기수별 학생 명단란에서는 기수별로 Root 소속 학생들을 확인 가능합니다.
